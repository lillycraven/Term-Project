Our Conclusion

Throughout this project, we both got to explore our passion and interest for data analytics as well as personal finance. We were able to analyze a set of financial data from an Amazon order history to provide us insights on our spending and where we might be able to cut back. We utilized libraries and methods we had never used before and felt that this project challenged us to think outside the classroom. It took us over 2-3 weeks to complete including getting the data, cleaning, and outputting what we wanted it to look like as well as using Flask to create our very own website. Our HTML pages and Flask code was a huge learning curve; especially being able to showcase our images directly from running the code. We hope that this tool is helpful for those that have an abundance of data and want to get clearer insights on their order history.
